## Key
 | Name     | Bits (inclusive, 1-indexed) | Details                                              | 
 | -------- | --- | ---------------------------                                                  |
 | CHECKSUM | 9-16 | Self-explanatory.                                                           |
 | .        | 1-8, 17-22, 24-25, 29-36, 47-56 | Stays the same between messages. Can be ignored. |
 | F        | 42-44 | Fan speed                                                                  |
 | L        | -     | Length (generated by parser)                                               |
 | M        | 45-56 | Mode                                                                       |
 | s        | 23    | ~Swing                                                                     |
 | S        | 26-28 | Smart mode (whatever that is)                                              |
 | T        | 37-41 | Temperature                                                                |

### Fan Speeds
 - `010`: 1
 - `100`: 2
 - `101`: 3

### Modes
 - `01`: Cool
 - `10`: Dry
 - `11`: Fan
